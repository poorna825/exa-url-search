🚀 USAGE EXAMPLES
═══════════════════════════════════════════════════════════════════════════

API:
  # Single domain
  GET /api/search?query=cooking&domains=tiktok

  # Multiple domains
  GET /api/search?query=AI&domains=youtube,reddit,medium

  # List domains
  GET /api/domains

Frontend:
  1. Visit http://localhost:5173/
  2. Select platforms (e.g., YouTube + Dev.to)
  3. Search: "python tutorial"
  4. Get results from both platforms instantly!
