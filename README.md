# exa-url-search

A flexible search engine that retrieves URLs of searched topics from various domains, starting with TikTok. Easily extensible to support multiple platforms and content sources.

## Project Structure

- `my-search-frontend/` — React frontend
- `my-search-backend/` — Python backend

## Getting Started

### Frontend

1. Navigate to `my-search-frontend`
2. Install dependencies:  
   `npm install`
3. Start development server:  
   `npm run dev`

### Backend

1. Navigate to `my-search-backend`
2. Install dependencies (if any):  
   `pip install -r requirements.txt`
3. Run the backend:  
   `python main.py`

## Deployment

- Frontend: Deploy on Vercel
- Backend: Deploy on Render, Railway, or similar

## Environment Variables

Store API keys and secrets in `.env` files.  
Do not commit `.env` files to the repository.
